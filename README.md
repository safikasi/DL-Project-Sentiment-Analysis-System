This repository contains tutorials and implementations for **Sentiment Analysis using Deep Learning models in PyTorch**.  
Developed as part of a **6th Semester Deep Learning course**, this project focuses on **sequence classification techniques** to predict sentiment from movie reviews.

> ⚠️ If you find any mistakes or have suggestions, feel free to open an issue.

## 🚀 Getting Started

Install dependencies:

```bash
pip install -r requirements.txt --upgrade

Make sure you are using Python 3.9+ and have PyTorch installed.

## 📚 Tutorials

###🔹 1 - Neural Bag of Words

This tutorial covers the workflow of a sequence classification project using a Neural Bag-of-Words (NBOW) model.

Data preprocessing
Tokenization
Vocabulary building
Baseline model training

🔗 

### 🔹 2 - Recurrent Neural Networks (RNN / LSTM)

This section improves performance using RNNs, specifically LSTM models.

Sequential data handling
Context understanding
Long-term dependency learning

🔗 

###🔹 3 - Convolutional Neural Networks

This tutorial applies CNNs for text classification.

Feature extraction
Convolution filters
Efficient training

🔗 

###🔹 4 - Transformers (BERT)

This section demonstrates Transformer-based models, specifically BERT, for high-accuracy sentiment prediction.

Pre-trained models
Fine-tuning
State-of-the-art performance

🔗
